# Git Basics

Some basic Git Commands

## Cloning

***To clone a Repository***
````
git clone https://github.com/anannyadutta09/PythonTests.git
````

## Pushing files

***Add files to git staging***
````
git add *
````

***Add comment and commit***
````
git commit -m "MEANINGFUL MESSAGE"
````

***Push the changes***
````
git push
````
